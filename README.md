<p align="center">
	Client iOS non-officiel pour <a href="https://www.izly.fr">Izly</a>, le portefeuille électronique étudiant français.
</p>

## Fonctionnalités

- Connexion sécurisée avec stockage des identifiants dans le Keychain
- Solde en temps réel avec graphe interactif de l'évolution
- Historique des transactions (paiements, rechargements, virements)
- Activation du compte via lien e-mail
- Refresh de session via OTP (HOTP)
- Thème personnalisable (couleur d'accentuation, apparence claire/sombre)

## Stack

- **SwiftUI** — interface
- **Swift Charts** — graphe du solde
- **Keychain** — stockage sécurisé des credentials
- API **SOAP/REST Izly** — données du compte
